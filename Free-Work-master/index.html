Теги strong, b, em и i

<strong> - ВАЖНОСТЬ;
<em> - ИНТОНАЦИЯ.

Перенос и разделитель

<br> - ПЕРЕНОС
<hr> - ИНТОНАЦИЯ

Цитаты

<blockquote> - выделение длинных цитат. Несколько абзацев.
<q> - выделение коротких цитат. Обрамляется в кавычки.
<cite> - для выделения источника цитаты, название произведения, автора цитаты.

Верхние и нижние индексы

<sup> - верхний индекс.
<sub> - нижний индекс.

Пометка изменений в документе

<del> - выделяет текст, который был удален в новой версии документа.
Пример:
<del>вредная привычка</del> - вредная привычка
>ins> - выделяет текст, который был добавлен в новой версии документа.
Пример:
<ins>Какой-то текст</ins> - какой-то текст

!АТРИБУT INS! - datetime

Пример:
<ins datetime="2018-03-28">какой-то текст </ins>

<pre>

<pre> - отображение и учитывание всех пробелов в HTML документе.

<mark>

<mark> - выделение текста с желтым фоном

Ссылки

<a href="ссылка">Отображающийся текст</a> - Отображающийся текст

Относительные адреса

      ./courses или ../../couses

Ссылка с якорем

<a href="#ссылка">Текст ссылки</a>
Пример:
<a href="#title">Нужно запомнить вот это</a>
В пустом, нужном месте в HTML-коде вставляем таким образом:
<h2 id="title">Какой-то текст</h2>

Рамка с помощью CSS

Пример:
        table {
      border: 3px solid цвет;
   }

Улучшение отображения рамок

Пример:
      table {
      border-collapse: collapse;
    }

Горизонтальные и вертикальные рамки

Задаются с помощью:
  • border-top
  • border-right
  • border-bottom
  • border-left

Отступы внутри ячеек

Пример: 
          td {
          padding: 10px;
          padding-right: 20px;
        }

Отступу между ячейками

В HTML:
<table cellspacing="значение(цифра)"></table>
В CSS: 
            table {
            border-spacing: 5px;
          }
